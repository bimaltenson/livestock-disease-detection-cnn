# Livestock Disease Detection Using CNN

## Project Overview
This project focuses on detecting livestock diseases from cattle images using Convolutional Neural Networks (CNN) and MobileNetV2.

## Dataset
Dataset Source:
https://www.kaggle.com/

Classes:
- Healthy
- Lumpy Skin Disease
- Foot-and-Mouth Disease

Total Images: 3244

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn

## Project Structure
- Proposal Document
- Kaggle Notebook
- Output Figures
- Dataset Information

## Results
CNN Accuracy: 78.00%
MobileNetV2 Accuracy: 37.19%

## How to Run
1. Download the dataset.
2. Open the notebook in Kaggle.
3. Run all cells.
4. Generated figures will be saved automatically.
