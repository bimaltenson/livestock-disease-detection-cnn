# Livestock Disease Detection Using CNN

## Project Overview

This project focuses on detecting livestock diseases from cattle images using Convolutional Neural Networks (CNNs). The objective is to classify cattle images into different health conditions and evaluate the effectiveness of deep learning techniques for disease detection.

## Dataset Information

The dataset contains cattle images belonging to three classes:

| Class                  | Number of Images |
| ---------------------- | ---------------: |
| Healthy                |            1,291 |
| Lumpy Skin Disease     |            1,207 |
| Foot-and-Mouth Disease |              746 |
| Total                  |            3,244 |

The dataset was collected from Kaggle and used for multi-class image classification.

## Research Questions

1. Can a CNN accurately classify livestock diseases from image data?
2. How effectively can the model distinguish healthy cattle from diseased cattle?
3. Which disease category achieves the highest classification performance?
4. How does a custom CNN compare with MobileNetV2?
5. What classification challenges occur during disease prediction?
